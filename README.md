# epinephrine
In-memory database


## Installation
```
pip install epinephrine
```

Type `epinephrine` then press enter. Server will start up.

## Usage

Try `telnet localhost:9999`

All commands starts with `#`.

```
#INSERT:your data1
#INSERT:your data2
#RETRIVE:2:0
```


## Commands

### insert
INSERT:{text}

### retrive
RETRIVE:{line}:{page}
